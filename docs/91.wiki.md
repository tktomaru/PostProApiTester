# PostPro API Tester - 用語集・略語一覧

## 1. 基本用語

| 用語 | 定義 |
|------|------|
| **API (Application Programming Interface)** | アプリケーション間通信のためのインターフェース |
| **Chrome拡張機能** | Google Chromeブラウザに機能を追加するプログラム |
| **コレクション** | 関連するHTTPリクエストのグループ |
| **シナリオ** | 複数のリクエストを順次実行するワークフロー |
| **エコーAPI** | 送信されたデータをそのまま返すAPI (https://reply.tukutano.jp) |
| **環境** | 開発/ステージング/本番等のコンテキストを表す変数セット |
| **変数置換** | {{variableName}}を実際の値に置き換える機能 |

## 2. HTTPリクエスト・レスポンス関連

| 用語 | 定義 |
|------|------|
| **HTTPメソッド** | GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS |
| **リクエストヘッダー** | HTTPリクエストと共に送信されるメタデータ |
| **クエリパラメータ** | URLに付加されるパラメータ |
| **リクエストボディ** | HTTPリクエストで送信されるデータペイロード |
| **レスポンスステータス** | HTTPステータスコード (200, 404, 500等) |
| **レスポンス時間** | リクエスト完了までの所要時間 |
| **JSONPath** | JSON構造からデータを抽出するためのクエリ言語 |
| **CORS (Cross-Origin Resource Sharing)** | クロスオリジンリクエストを制御するセキュリティ機能 |

## 3. 認証・セキュリティ

| 用語 | 定義 |
|------|------|
| **認証 (Authentication)** | アイデンティティを検証するプロセス |
| **Basic認証** | ユーザー名/パスワードを使用するシンプルな認証方式 |
| **Bearerトークン** | Authorizationヘッダーで渡される認証トークン |
| **APIキー** | API認証のための秘密鍵 |
| **OAuth2** | オープンな認証・認可の標準規格 |
| **AES-256** | 256ビット暗号鍵による高度暗号化標準 |
| **PBKDF2** | パスワードベース鍵導出関数 |
| **暗号化** | セキュリティのためデータを符号化するプロセス |
| **サニタイゼーション** | 入力データをクリーンにするプロセス |

## 4. Chrome拡張機能関連

| 用語 | 定義 |
|------|------|
| **バックグラウンドスクリプト** | 拡張機能のバックグラウンドコンテキストで実行されるスクリプト |
| **コンテンツスクリプト** | Webページのコンテキストで実行されるスクリプト |
| **ポップアップUI** | 拡張機能アイコンクリック時に表示されるメインインターフェース |
| **Chrome Storage API** | Chrome拡張機能での永続データストレージAPI |
| **メッセージパッシング** | 拡張機能コンポーネント間の通信メカニズム |
| **注入スクリプト** | ページの実行コンテキストに挿入されるスクリプト |
| **Service Worker** | バックグラウンドワーカースレッド |
| **Manifest V3** | Chrome拡張機能の最新バージョン仕様 |
| **Sandbox** | セキュリティのための分離実行環境 |

## 5. テスト・検証

| 用語 | 定義 |
|------|------|
| **テストスクリプト** | HTTPレスポンス受信後に実行される検証用JavaScript |
| **Pre-requestスクリプト** | HTTPリクエスト送信前に実行されるJavaScript |
| **テストアサーション** | 期待値と実際の結果をチェックする検証文 |
| **単体テスト** | 個別コンポーネントの分離テスト |
| **統合テスト** | コンポーネント間の相互作用テスト |
| **E2E (End-to-End)** | エンドツーエンドテスト |
| **モック** | テスト用のシミュレートオブジェクト |
| **スタブ** | テスト用の簡略化実装 |
| **テストカバレッジ** | テストされたコードの割合 |

| **VariableManager** | 3階層変数システム（グローバル/環境/コレクション）を扱うコンポーネント |
| **ScriptExecutor** | ユーザースクリプトの安全実行コンポーネント |

| **StateManager** | アプリケーション全体の状態管理コンポーネント |
| **ResponseViewer** | HTTPレスポンスデータ表示UIコンポーネント |
| **RequestBuilder** | HTTPリクエスト構築UIコンポーネント |
